# Question Bank

##### This is a sample just project for ...

### Prerequisit

- Docker
- Docker compose
- JDK 17
- node 21

### Commands

Run the Backend & Frontend:

```
docker compose up --build
```

Run frontend only (from packages/app folder),

```
yarn up
```

or

```
yarn dev
```

Run backend only (from packages/api)

```
mvn spring-boot:run
```
